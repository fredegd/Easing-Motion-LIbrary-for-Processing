    .getSpan()


    Description

    This function returns the current span value of an EasyEase Object.


    Examples

    getSpan_01.pde
    import easy.ease.*;

EasyEase curve = new EasyEase(this, 4.0, 2.0, 1.5, 0.0);

void setup() {
  size(600, 400);
  println(curve.getSpan());
}

// prints 1.5 to the console


    getSpan_02.pde
    import easy.ease.*;

EasyEase curve = new EasyEase(this);

void setup() {
  size(600, 400);
  println(curve.getSpan());
}

// prints 1.0 to the console


    Syntax

    myCurve.getSpan(intensity)


    Return

    float
