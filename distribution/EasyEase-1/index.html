Download

Download Easy Ease - A collection of utility functions for Easing motions and Nonlinear distribution version 1.0.0 (1) in .zip format.

Installation

Unzip and put the extracted EasyEase folder into the libraries folder of your Processing sketches. Reference and examples are included in the EasyEase folder.

Keywords. easing motion easing-curve gradient non-linear curve

Reference. Have a look at the javadoc reference here. A copy of the reference is included in the .zip as well.
